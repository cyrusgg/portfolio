Dynamics Controller for DIGIT



The TRACE lab at UMASS Lowell is currently working on creating a controller for DIGIT, a humanoid bipedal robot by Agility Robotics. The end goal is a create a controller for DIGIT which can be implimented on hardware. Important _____ of this project include discrete dynamics systems, error dynamics, FROST, Forward kinematics, holonomic constraints, and phase variable and trajectory planning/optimization.
